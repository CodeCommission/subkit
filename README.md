# Subkit

> GraphQL APIs made fast and easy.

## Setup & Run

Firstly, install it:

```bash
npm i -g subkit
```

Create your first GraphQL API:

```bash
subkit create my-graphql-api
```

Serve locally:

```bash
cd my-graphql-api
subkit serve
```
